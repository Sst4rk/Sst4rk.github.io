TCLEAN

Tclean

Plataforma: Windows

Lenguaje: Python

Tipo: Instalable (No Portable)

DESCRIPCION

Herramienta creada en Python para la plataforma Windows, con el objetivo de simplificar la tarea de limpiar los archivos temporales del sistema.

Esta herramienta, crea un icono en la bandeja del sistema para que con un simple click, puedas eliminar todos los archivos temporales, que genera el sistema y que al generar y no eliminarse automaticamente realentizan al mismo sistema.

Con el menu integrado que se desplega con el Click Derecho puedes hacer que la herramienta se ejecute al iniciar Windows.
